Help is here.

We have many resources, both official and community-based, to help you with any issues you encounter while using Gamecraft. For any inquiries not answered by the resources below, you can also contact us directly via the email form.

Documentation

Our documentation is the prime destination for the fine details of any Gamecraft feature. Here you can see the available logic blocks as well as which difficulty level they belong to, real-life usage examples, and answers to common questions. We recommend that you consult the documentation first - you can find it here. If you find that the docs are lacking for any reason, or have an idea for improvement, let us know using the contact form below.

Forums

We host official forums, where Gamecraft newcomers and experts alike are welcome to discuss their ideas and support each other throughout the sometimes grueling task of videogame development. We also have in-house Gamecraft experts answering questions posted here. Before creating a new thread, try to use the search function first - it's possible that someone else had the same problem, and you'll be able to get it resolved right away! You can visit the forums through here.

Contact form

If your questions were not answered by the resources above, or you are looking to contact us directly for any reason, use the form below to get in touch with us. You will receive an answer within 2 working days.
